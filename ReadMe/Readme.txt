After run WAMP Server, follow the following steps

1st Step: Download IMS folder from drive
2nd Step: Copy the main project folder
3rd Step: Paste in wamp64/www
4th Step: Open a browser and go to URL “http://localhost/phpmyadmin/”
5th Step: Then, click on the databases tab
6th Step: Create a database naming “project” and then click on the import tab
7th Step: Click on browse file and select “project.sql” file which is inside the “database” folder
8th Step: Click on go.
After Creating Database,

9th Step: Open a browser and go to URL “http://localhost/IMS/login.html”

For the admin panel, go to URL “http://localhost/IMS/adminLoginPage.php/”
For the teacher panel, go to URL “http://localhost/IMS/teacherLoginPage.php/”
For the teacher panel, go to URL “http://localhost/IMS/studentLoginPage.php/”
